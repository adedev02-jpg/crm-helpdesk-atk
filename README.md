## 📋 Deskripsi Proyek

**CRM Helpdesk ATK Sekolah** adalah sistem manajemen layanan pelanggan berbasis web yang dirancang untuk CV. Indah Maju Gemilang — distributor alat tulis kantor (ATK) untuk sekolah-sekolah di wilayah Jawa Barat dan sekitarnya.

Sistem ini merupakan solusi digitalisasi layanan pelanggan yang sebelumnya dilakukan melalui:
- 🛒 **SIPLaH** — pengadaan barang resmi sekolah
- 💬 **WhatsApp Business** — komunikasi pesanan
- 🌐 **Website** — katalog produk
- 📸 **Instagram** — pemasaran dan promosi

---

## 🎯 Tujuan Proyek

Membangun platform CRM terpadu yang memungkinkan:
1. Pelanggan (sekolah) mengajukan pengaduan secara terstruktur
2. Admin memantau dan merespons tiket secara efisien
3. Manajemen memperoleh laporan performa layanan secara real-time

## ✨ Fitur Utama

### 👨‍💼 Fitur Pelanggan
| Fitur | Deskripsi |
|-------|-----------|
| 🏠 Dashboard | Ringkasan status tiket aktif |

| 📋 Riwayat Tiket | Histori semua pengaduan |

### 🔐 Fitur Admin
| Fitur | Deskripsi |
|-------|-----------|
| 📊 Dashboard Admin | Overview semua tiket & performa |
| 📥 Tiket Masuk | Kelola & respons pengaduan |
| 🏫 Data Pelanggan | Database sekolah terdaftar |
| 📈 Laporan | Grafik & analitik layanan |

## 🗂️ Kategori Tiket

- 📦 **Produk** — Keluhan terkait kualitas/kuantitas barang
- 🚚 **Pengiriman** — Keterlambatan atau kesalahan pengiriman
- 💳 **Pembayaran** — Konfirmasi pembayaran & invoice

## 🚀 Cara Menjalankan

## 🛠️ Teknologi yang Digunakan

| Teknologi | Keterangan |
|-----------|------------|
| HTML5 | Struktur halaman |
| CSS3 | Styling & layout (tanpa framework) |
| JavaScript | Interaktivitas & navigasi antar halaman |
| Google Fonts | Tipografi (Inter + Plus Jakarta Sans) |

> Tidak menggunakan framework eksternal — murni HTML, CSS, dan JavaScript vanilla.
